## Notes

- **Minimum recommendation:** Always include iLO Advanced + Foundation Care 24x7
- **RAID controller:** Smart Array P408i-a standard; P816i-a for large arrays (+45,000)
- **Redundant PSU:** Add ~15,000 for redundant power supply (strongly recommended)
- **Rails:** Quick deploy rails included; cable management arm extra (~8,000)
- **Windows Server licensing:** Not included; budget ~80,000-1,50,000 per VM host
- **VMware:** vSphere Standard ~1,20,000 per CPU; Essentials Plus kit ~3,50,000 (6 CPU max)
